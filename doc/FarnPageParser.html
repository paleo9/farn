    class FarnPageParser

    Class FarnPageParser Parses and stores the information from the resulting page of a search performed on their website and subsequently stored by the user.

    Attributes

    headings[R]

    Headings common to all searches.

    name[R]

    The kind of part e.g. ‘Through-hole resistors’

    properties_headings[R]

    Headings for the set of part properties. They vary, depending on part type e.g. resistors will include ‘Resistance’. These may be used as an argument to rows[] row to_hash.

    query_date[R]

    The date on which the search was performed.

    rows[R]

    An array containing each row from the results table.

def to_hash
  hash = { name: @name, query_date: @query_date }
  hash[:rows] = []
  @rows.each_with_index do |r, i|
    hash[:rows] << r.to_hash(@properties_headings)
  end
  hash
end

def to_s
  s = "Name: #{@name} Query date: #{query_date} Headings: "
  s += headings.to_s
  s += properties_headings.to_s
  rows.each { |r| s += r.to_s }
  s
end
